=================================================
Pasos para instalar la aplicacion Alohandes
=================================================

1. Haga un fork o clone el repositorio a su sistema
2. Desde el IDE de su preferencia, abra la carpeta que contiene
el repositorio, mas especificamente, abra la carpeta
llamada alohandes
4. En la carpeta main -> resources -> META-INF abra el archivo
persistence e inserte las credenciales de su usuario en la parte
de username y password
para acceder al host de la base de datos
5. Ejecute la clase InterfazAlohandesApp.java en la carpeta
src para acceder
a la interfaz y poder hacer uso de la aplicacion de manera
exitosa
6. En caso de querer ver
la demostracion de como funciona la aplicacion
ejecute la clase InterfazAlohandesDemo.java en la carpeta src
7. En caso de querer limpiar o ver las bitacoras del esquema,
seleccione el boton de 'manenimiento' y ahi podra hacer uso
de todas las funcionalidades del log y otros
8. En el boton 'Acerca de' encuentra la informacion de las
licencias y permisos del aplicativo para el desarrollo 
del curso sistemas transaccionales.
